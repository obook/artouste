# Tableau de bord de l'Alouette II - documents de reference

Ce dossier rassemble les documents qui ont servi de reference pour modeliser la
planche de bord (le tableau de bord, ou "panel") de l'Alouette II dans Artouste.
Ils sont fournis a titre documentaire et pedagogique.

## Fichiers

| Fichier | Description |
|---------|-------------|
| `board.jpg` | Photographie du tableau de bord reel d'une Alouette II. |
| `al2m2.jpeg` | Manuel de vol, figure 2-5 : "Instrument panel and control pedestal" (appareils n. 1327 et anterieurs), page 2-13. |
| `al2m2b.jpeg` | Manuel de vol, figure 2-5a : meme planche pour les appareils n. 1328 et ulterieurs, page 2-13b. |
| `al2m3a.jpeg` | Manuel de vol, legende de la figure 2-5a : les 71 reperes des instruments et commandes, page 2-13a. |
| `al2m10.jpeg` | Manuel de vol, appendice 1, figure 1-1 : "Instrument markings" (graduations des cadrans : badin, pas collectif, double tachymetre, jauge carburant, indicateur triple). |
